## 📋 Lo que se ha implementado

### Frontend Angular - COMPLETO ✅

Se ha creado una aplicación Angular completa con:

#### 📦 Estructura del Proyecto
- Angular 17
- TypeScript 5
- Bootstrap 5 y Bootstrap Icons
- Angular Router
- HttpClient Module

#### 🧩 Componentes Creados
- Navbar (barra de navegación)
- Footer
- Home (página de inicio)
- FiguraList (catálogo con cards y tabla, filtros, paginación)
- FiguraDetail (detalle de figura)
- FiguraForm (crear/editar figura)

#### 🔧 Servicios
- FiguraService: CRUD completo contra la API REST

#### 📐 Modelos (Interfaces TypeScript)
- Figura (modelo principal)

#### 🔀 Routing
- `/` → Home
- `/figuras` → Catálogo
- `/figuras/:id` → Detalle
- `/figuras/nueva` → Crear figura
- `/figuras/editar/:id` → Editar figura

#### ✨ Funcionalidades
- CRUD completo: crear, leer, actualizar, eliminar figuras
- Paginación y filtros (por anime/categoría y búsqueda de texto)
- Validaciones en formularios reactivos
- Spinners de carga
- Mensajes de éxito/error
- Responsive con Bootstrap
- Confirmación antes de eliminar
- Animaciones y efectos hover
- Bootstrap Icons
- Navegación intuitiva

---

## 🚀 Cómo Iniciar el Proyecto

```bash
cd frontend-angular
npm install
npm start
# Abre http://localhost:4200
```
