# Frontend React

Cliente React minimal que consume la API del `backend` para mostrar la colección de figuras.

Contenido del ejemplo:
- `src/`: código fuente React.
- `src/App.jsx`: componente principal que hace llamadas a `/api/figures`.
- `src/components/FigureList.jsx`: muestra una lista de figuras.

Cómo arrancar:

1. `cd frontend-react`
2. `npm install`
3. `npm run dev`

Nota para el profesor: este cliente es un punto de partida; se puede ampliar con formularios, rutas y autenticación.
